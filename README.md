### 使用方法

在谷歌浏览器地址栏输入 <chrome://extensions> 进入谷歌浏览器扩展程序页面。\
将该页面右侧的`开发者模式`打开。\
点击`加载已解压的扩展程序`按钮，选择`eazy-bookmark`目录。\
添加完成后，将开关打开。\
地址栏右侧点击 🧩，就可以找到`简单书签`的图标了。
